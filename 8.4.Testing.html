What is Software Testing?

Software Testing is the process of executing a program with the intent finding errors.
It is a process of verifying are we developing the right product? And Validating does the Developed product is right?

Software testing = Verification + Validation.

Verification: It is a process of verifying are we developing the right product or not. This is called as static Testing.

Validation: It is a process of validating does the developed product is right or not. This is called as dynamic Testing.

Importance / Objectives of Testing

  1. To make sure that the application is developed as per the customer requirements or not.

  2. Finding defects.

  3. To make sure all problems are resolved and close.

  4. Software Testing is helpful to deliver a quality product and risk-free product to the customer.
